# Simulation of DDoS attack in SDN Network using Floodlight Controller and Mininet

```

1). cd floodlight
    java -jar target/floodlight.jar

2). sudo mn --controller=remote,ip=127.0.0.1,port=6653 --topo=single,3

3). cd sFlow-rt
    sudo ./start.sh

4). sudo ovs-vsctl 


```
